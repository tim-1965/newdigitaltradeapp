# Trade Digitalisation Benefits Calculator

## All Features Included

✅ **Annual volume in millions** - Enter 200 for $200M (not 200000000)
✅ **Interactive sliders** - All inputs except annual volume use sliders
✅ **Debounced calculations** - Only updates when you release the slider
✅ **Compact horizontal layout** - Sliders next to labels
✅ **Combined results card** - P&L and Working Capital in one card
✅ **Aligned right panel** - Results align with input boxes
✅ **Natural scrolling** - No sticky positioning

## Right Panel Layout

The result cards are positioned to align with the left panel input boxes:

1. **Combined Results Card** (at top) - P&L Benefit + Working Capital
2. **Working Capital Benefits** → aligns with Box 2 (Early Payment section)
3. **Early Payment Details** → below Working Capital
4. **Headcount Efficiency** → aligns with Box 3 (AP Headcount)
5. **P&L Benefits After Costs** → aligns with Box 4 (Customs)

## Installation

```bash
npm install
npm run dev
```

## Quick Start

1. Enter your annual volume in millions (e.g., 200 for $200M)
2. Adjust sliders to match your company's parameters
3. View real-time results in the right panel
4. Switch to "Simulation" tab to see before/after financials

## Key Features

- **Millions input**: Just type 200 instead of 200000000
- **Smooth sliders**: Drag without lag, calculations run on release
- **Visual alignment**: Results positioned next to relevant inputs
- **Auto-save**: Your inputs persist between sessions
- **Compact design**: More information visible at once

All calculations match the Excel spreadsheet formulas exactly.
